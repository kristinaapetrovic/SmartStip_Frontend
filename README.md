# Student Scholarship Portal

A React frontend for managing student scholarships. Supports roles: **student**, **committee**, and **administrator**.

## Backend Models

- Student, Committee, Administrator
- Scholarship Call, University, Faculty, Location
- Contract, Application

## Tech Stack

- React 18 + Vite 6
- React Router DOM 7
- Bootstrap 5 + React Bootstrap
- Axios (configured in `src/axios/`)

## Scripts

- `npm run dev` - Start dev server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
